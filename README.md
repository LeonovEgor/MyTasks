# MyTasks
Курсовой проект на курсе Geekbrains Kotlin.   
Язык: Kotlin

Позволяет:  
- создавать разноцветные заметки
- отображать заметки в виде списка
- сохранять заметки

## Стек технологий:  
- Coroutines
- Sqlite;
- Matherial Design;
